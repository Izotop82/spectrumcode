Программирование на Python

Полный курс для начинающих от IT-школы SPECTRUM

О курсе

Этот курс познакомит вас с основами программирования на Python - одном из самых популярных языков в мире. Вы научитесь писать программы, работать с данными и решать реальные задачи.

Курс разработан специально для школьников 12-14 лет и включает 12 ключевых тем, от основ до более сложных концепций.

Содержание курса

Основы Python

  • Ввод и вывод данных - работа с input() и print()
  • Арифметика - операции с числами
  • Условные конструкции - if, elif, else

Циклы и функции

  • Цикл с параметром - for и range
  • Цикл с условием - while
  • Функции - создание и использование

Работа с данными

  • Строки и срезы - обработка текста
  • Методы и функции строк - полезные инструменты
  • Работа с файлами - чтение и запись

Продвинутые темы

  • Обработка ошибок - try/except
  • Работа с модулями - import
  • Списки - коллекции данных
