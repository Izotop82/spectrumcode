Python: Цикл while

Выполнение действий пока условие истинно

Цикл while в Python

Цикл while выполняет блок кода, пока заданное условие остается истинным (True). Это мощный инструмент для создания повторяющихся действий с неизвестным количеством итераций.

Базовый синтаксис цикла while

Цикл while имеет следующий синтаксис:

# Простейший пример цикла while
count = 0
while count < 5:
    print(f"Итерация №{count}")
    count += 1  # Не забываем изменять условие!

print("Цикл завершен")
Итерация №0
Итерация №1
Итерация №2
Итерация №3
Итерация №4
Цикл завершен
Важно: Всегда убедитесь, что условие цикла while в какой-то момент станет ложным (False), иначе цикл будет выполняться бесконечно!

Бесконечные циклы и их прерывание

Иногда бесконечные циклы полезны, но важно предусмотреть условие выхода:

# Пример бесконечного цикла с выходом по условию
total = 0
while True:  # Бесконечный цикл
    num = int(input("Введите число (0 для выхода): "))
    if num == 0:
        break  # Выход из цикла
    total += num

print(f"Сумма введенных чисел: {total}")
(После ввода 5, 3, 2, 0)
Сумма введенных чисел: 10

Операторы break и continue

Как и в цикле for, в while можно использовать:
break - немедленный выход из цикла
continue - переход к следующей итерации

# Пример break и continue
number = 0
while number < 10:
    number += 1
    if number % 2 == 0:
        continue  # Пропускаем четные числа
    if number == 7:
        break     # Выходим при достижении 7
    print(number)

Цикл while с else

В Python цикл while может иметь блок else, который выполняется, если цикл завершился нормально (без break):

# Пример while с else
attempts = 3
password = "qwerty"

while attempts > 0:
    user_input = input("Введите пароль: ")
    if user_input == password:
        print("Доступ разрешен!")
        break
    attempts -= 1
    print(f"Неверно! Осталось попыток: {attempts}")
else:
    print("Доступ запрещен!")
(После 3 неверных попыток)
Доступ запрещен!

Практические применения while

Цикл while часто используется для:

  • Обработки пользовательского ввода
  • Ожидания определенного события
  • Работы с данными неизвестного размера
  • Создания игровых циклов
# Пример: меню программы
while True:
    print("\n1. Показать список")
    print("2. Добавить элемент")
    print("3. Удалить элемент")
    print("4. Выход")
    
    choice = input("Выберите действие: ")
    
    if choice == "1":
        print("Список элементов...")
    elif choice == "2":
        print("Добавление элемента...")
    elif choice == "3":
        print("Удаление элемента...")
    elif choice == "4":
        print("Выход из программы")
        break
    else:
        print("Неверный ввод!")

Практические задания

Задание 1: Угадай число

Напишите игру, где:
1. Программа загадывает случайное число от 1 до 100
2. Пользователь пытается угадать число
3. Программа подсказывает "больше" или "меньше"
4. Игра продолжается, пока число не угадано

import random

secret = random.randint(1, 100)
attempts = 0

while True:
    guess = int(input("Угадайте число от 1 до 100: "))
    attempts += 1
    
    if guess == secret:
        print(f"Поздравляем! Вы угадали за {attempts} попыток")
        break
    elif guess < secret:
        print("Загаданное число больше")
    else:
        print("Загаданное число меньше")

Задание 2: Калькулятор

Напишите калькулятор, который:
1. Предлагает выбрать операцию (+, -, *, /)
2. Запрашивает два числа
3. Выводит результат
4. Спрашивает, хочет ли пользователь продолжить

while True:
    print("\nДоступные операции:")
    print("+ - сложение")
    print("- - вычитание")
    print("* - умножение")
    print("/ - деление")
    
    op = input("Выберите операцию (или 'q' для выхода): ")
    if op == 'q':
        break
        
    a = float(input("Первое число: "))
    b = float(input("Второе число: "))
    
    if op == '+':
        print(f"Результат: {a + b}")
    elif op == '-':
        print(f"Результат: {a - b}")
    elif op == '*':
        print(f"Результат: {a * b}")
    elif op == '/':
        if b == 0:
            print("Ошибка: деление на ноль!")
        else:
            print(f"Результат: {a / b}")
    else:
        print("Неизвестная операция!")

Задание 3: Проверка пароля

Напишите программу, которая:
1. Запрашивает у пользователя пароль
2. Проверяет, что пароль содержит минимум 8 символов
3. Проверяет, что пароль содержит цифры
4. Дает 3 попытки на ввод корректного пароля

attempts = 3

while attempts > 0:
    password = input("Придумайте пароль (мин. 8 символов, с цифрами): ")
    
    if len(password) < 8:
        print("Пароль слишком короткий!")
    elif not any(char.isdigit() for char in password):
        print("Пароль должен содержать цифры!")
    else:
        print("Пароль принят!")
        break
        
    attempts -= 1
    print(f"Осталось попыток: {attempts}")
else:
    print("Попытки исчерпаны!")
